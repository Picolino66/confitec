# Confitec
Front-End da aplicação Confitec.

## Requisitos
Esse projeto foi construído com as seguitnes versões:
Angular CLI: 14.2.5
Node: 16.16.0
Package Manager: npm 8.19.2 

## Instalação
Na pasta do projeto:
npm install

## Configuração
Na pasta environments coloque a url do servidor no arquivo environments.ts
